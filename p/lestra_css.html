Документация по фрэймворку сайта

В основном делается для себя. Актуально на 17.05.2020. Фрэймворк подключается кодом:

<link rel="stylesheet" href="https://lestrangeqq.github.io/css/style.css">
    Шрифт:
  • .f-small {font-size: 0.8em;} - мелкий шрифт.
  • .f-medium {font-size: 1.2em;} - средний шрифт, чуть крупнее того, который выставлен в браузере пользователя.
  • .f-large {font-size: 1.5em;} - крупный шрифт.
  • .f-xlarge {font-size: 2.0em;} - очень крупный шрифт.
    Позиционирование объектов:
  • .f-right {float:right;} - обтекание по правому краю.
  • .center {text-align:center;} - выравнивание по центру.
  • .marg-side-10 {margin:0px 10px;} - небольшие отступы по краям.
  • .marg-10 {margin:10px 0px;} - небольшие отступы сверху/снизу.
  • .marg-20 {margin:20px 0px;} - средние отступы сверху/снизу.
  • .padd-5 {padding:5px;} - маленькие внутренние отступы.
  • .padd-10 {padding: 10px;} - средние внутренние отступы.
    Списки:
  • .ul-square {list-style-type: square;} - квадраты у элементов в UL списке.
  • .ul-circle {list-style-type: circle;} - круги у элементов в UL списке.
  • .ul-arrow {list-style-type: disclosure-closed;} - стрелки у элементов в UL списке.
    Прочее:
  • .clickable - по наведению на объект меняет курсор, как будто на этот элемент можно кликнуть.
