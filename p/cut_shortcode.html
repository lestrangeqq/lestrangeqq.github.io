Удаление шорткода из превью статьи

Иногда в превью попадает тег шорткода, и выглядит это некрасиво. Чтобы не перемещать его куда то ниже, убрать его можно таким образом:

var catchContents = document.querySelector('.home-feature');
for (var i = 0; i < 9; i++) {
catchContents.innerHTML = catchContents.innerHTML.replace('[contents]', '');
}

Где .home-feature - селектор блока, где скрипт будет искать и удалять шорткод, [contents] - сам шорткод который нужно удалить.
