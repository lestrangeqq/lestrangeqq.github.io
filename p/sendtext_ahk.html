Отправка текста по нажатию F2

Если есть необходимость отправлять определенный текст много раз, а команду ctrl-v использовать не хочется. Код:

F2::
  Send Hello World
Return

Если вы не знакомы с Autohotkey то вам следует:

  1. установить Autohotkey
  2. сохранить приведенный код в файле формата .ahk, например send_text.ahk
  3. в файле send_text.ahk заменить текст Hello World на свой.
  4. при необходимости изменить кнопку F2 на другую.
  5. запустить файл send_text.ahk
