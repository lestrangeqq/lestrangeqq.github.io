Отображение блока по наведению на другой

Простой скрипт на jQuery который позволит создать нечто вроде тултипа, будет показывать блок когда наведешь на другой. HTML разметка:

<p>
text <span id="tooltip">tooltip<span class="tooltip-toggle">text tooltip</span></span>lalala
</p>

Оформление блока, CSS:

.tooltip-toggle {
  display: none;
  position: absolute;
  z-index: 99;
  background:red;
  padding:10px;
}

JS:

$("#tooltip").hover(function(){
    $('.tooltip-toggle').show();
},function(){
    $('.tooltip-toggle').hide();
});

Также понадобится подключить библиотеку jQuery.
