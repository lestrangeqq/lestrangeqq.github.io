Реализация простого спойлера

Спойлер с кнопкой открывания и сворачивания, HTML:

<button id="spoilerButton" onclick="openSpoiler()">Открыть</button>
<div id="spoiler" class="spoilerHidden">
  <p>HELLO</p>
  <button id="spoilerClose" onclick="closeSpoiler()">Закрыть</button>
</div>

CSS прост как 5 копеек:

.spoilerHidden {
  display: none;
}
.spoilerShow {
  display: block;
}
.spoilerButtonHide {
  display: none;
}
#spoiler {
  background: red;
}

JS:

function openSpoiler() {
  var spoiler = document.getElementById('spoiler');
  var spoilerButton = document.getElementById('spoilerButton');
  spoiler.classList.add('spoilerShow');
  spoilerButton.classList.add('spoilerButtonHide');
}
function closeSpoiler() {
  spoiler.classList.remove('spoilerShow');
  spoilerButton.classList.remove('spoilerButtonHide');
}
