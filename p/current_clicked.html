Работа с конкретно кликнутым элементом из массива

Собираем список из всех элементов p, добавляем цикл, функцию возможности кликнуть по ним через addEventListener, и 'this' - то есть будет учитываться конкретно кликнутый элемент:

  let x = document.getElementsByTagName('p');
  for (i = 0; i < x.length; i++) {
    x[i].addEventListener("click", function() {
		this.style.color = 'red';
    });
  }
