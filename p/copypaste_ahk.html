Замена биндов копипастинга на Autohotkey

Однажды мне очень надоело использовать ctrl-a ctrl-c и ctrl-v для того чтобы выделить, скопировать и вставить текст. Да и через некоторое время пальцы начали болеть. И я перебиндил эти команды на кнопки F1, F2, F3. Код прост:

#NoEnv  ; Recommended for performance and compatibility with future AutoHotkey releases.
; #Warn  ; Enable warnings to assist with detecting common errors.
SendMode Input  ; Recommended for new scripts due to its superior speed and reliability.
SetWorkingDir %A_ScriptDir%  ; Ensures a consistent starting directory.

F1::^a
F2::^c
F3::^v

Если вы не знакомы с Autohotkey то вам следует:

  1. установить Autohotkey
  2. сохранить приведенный код в файле формата .ahk, например copypaste.ahk
  3. запустить файл copypaste.ahk
  4. использовать новую комбинацию биндов: SHIFT+A = F1, SHIFT+C = F2, SHIFT+V = F3
