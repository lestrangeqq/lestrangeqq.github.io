Адаптивные 2 колонки

Две колонки одинакового размера, занимающие по 50% площади экрана (или родительского блока). На экранах меньше 600px встанут вертикально одна за одной.

Разметка:

<div class="left">левая колонка</div>
<div class="right">правая колонка</div>

CSS:

.left {background:red; float:left; width:50%;}
.right {background:green; float:right; width:50%;}

@media only screen and (max-width: 600px) {
.left, .right {width:100%;}
}

Я добавил расцветку для наглядности.

левая колонка
правая колонка
