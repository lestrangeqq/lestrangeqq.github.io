Появляющийся тултип по клику на элемент

Скрипт делает возможным отображение тултипа по клику на элемент. HTML разметка:

<p style="position:relative; z-index:1;">просто текст<span class="tooltip-link" onclick="openTooltip()">
кликнув сюда появится тултип</span><span id="toolTip" class="tooltip-hide">текст тултипа
<span onclick="removeTooltip()" class="closeTooltip">x</span></span></p>

Пояснения: position:relative; необходим чтобы дочерние элементы шли поверх родительского, то есть чтобы можно было вызвать функцию закрытия окна тултипа. Класс closeTooltip - стилизация кнопки закрытия окна тултипа(х). Дальше CSS:

#toolTip {
  position: absolute;
  width: 60%;
  left: 10px;
  z-index: 999;
  padding: 30px;
  font-size: 13px;
  line-height: 20px;
  text-align: left;
  background: #d5f4ff
}
.tooltip-hide {
  display: none
}
.tooltip-show {
  display: block
}
.closeTooltip {
  background: #c9c5c5;
  color: #fff;
  position: absolute;
  top: 0;
  right: 0;
  padding: 3px 5px;
  cursor: pointer
}
.tooltip-link {
  color: #0962ee;
  font-weight: 700;
  cursor: pointer
}

Тут в принципе все понятно. Можно избавиться от всего, кроме первых трёх, и переверстать по своему. Теперь JS:

function openTooltip() {
  var element = document.getElementById("toolTip");
  element.classList.add("tooltip-show");
  $(function() {
    $("#toolTip").click(function(e) {
      var offset = $(this).offset();
      var relativeX = (e.pageX - offset.left);
      var relativeY = (e.pageY - offset.top);
    });
  });
}
function removeTooltip() {
  var element = document.getElementById("toolTip");
  element.classList.remove("tooltip-show");
}

Также понадобится подключить библиотеку jQuery.
