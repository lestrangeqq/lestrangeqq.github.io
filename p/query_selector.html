Использование querySelectorAll по css классам

Загоняем в переменную divs все содержимое тегов с классом .ttt, а после используя if делаем что нужно, например при совпадении слов добавляем отступ:

var divs = document.querySelectorAll(".ttt");
Array.from(divs).forEach(function(div) {
  if (div.textContent.indexOf("здесь будет отступ слева") >= 0) {
    div.style.paddingLeft = "20px";
  }
  if (div.textContent.indexOf("и тут тоже") >= 0) {
    div.style.paddingLeft = "20px";
  }
});

Все варианты здесь имеют класс .ttt, но только два из них имеют отступ, те самые которые прописаны в скрипте и совпали по словам.
