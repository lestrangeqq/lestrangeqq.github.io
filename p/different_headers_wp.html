Разные файлы заголовков в зависимости от условия

В wordpress можно использовать множество шаблонов заголовков. Например если в определенной категории хочется использовать не стандартный header.php а другой, например header-banki.php это можно сделать так:

if (get_post_type() == 'banki') :     // можно использовать типы постов, категории и тд
    get_header('banki');
else :
    get_header();
