Расчет среднего времени на прочтение статьи

Данный скрипт выполняет функцию подсчета среднего времени, которое понадобится для прочтения статьи. Первым делом прописываем это в functions.php:

if ( ! function_exists( 'gp_read_time' ) ) {
function gp_read_time() {
$text = get_the_content( '' );
$words = str_word_count( strip_tags( $text ), 0, 'абвгдеёжзийклмнопрстуфхцчшщъыьэюяАБВГДЕЁЖЗИЙКЛМНОПРСТУФХЦЧШЩЪЫЬЭЮЯ' );
if ( !empty( $words ) ) {
$time_in_minutes = ceil( $words / 200
 );
return $time_in_minutes;
}
return false;
}
}

Следующий код выводит время на странице, прописать его можно к примеру в самой статье, то есть в файле single.php:

<?php echo gp_read_time(); ?>
<?php _e( 'мин', ' ' ); ?>
