Убиваем любой процесс с Autohotkey

Код:

F12::Process, Close, chrome.exe

F12 - кнопка по нажатию на которую процесс, взятый для примера, chrome.exe будет полностью закрываться.

Если вы не знакомы с Autohotkey то вам следует:

  1. установить Autohotkey
  2. сохранить приведенный код в файле формата .ahk, например kill_any_process.ahk
  3. запустить файл kill_any_process.ahk
  4. нажать F12 чтобы убить заданный в скрипте процесс.
